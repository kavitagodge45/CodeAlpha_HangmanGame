# CodeAlpha Hangman Game 

Hangman is a classic word guessing game developed using Python. The player has to guess the correct word by entering one letter at a time before running out of attempts.

# How the Game Works
- A random word is selected from a predefined list.
- The word is displayed as underscores (_ _ _ _).
- The player guesses one letter at a time.
- If the guess is correct, the letter is revealed.
- If incorrect, the number of attempts decreases.
- The player wins if the word is guessed before attempts run out.

# Features
- Random word selection
- Limited number of attempts (6 tries)
- Simple and interactive console interface
- Beginner-friendly logic

# Technologies Used
- Python
- Random module
- Loops and conditional statements

# How to Run
1. Install Python
2. Run the program:
   python hangman.py
3. Start guessing letters!
